# Bondify UI — Vite + React + TypeScript + Tailwind

A small frontend recreation of three screens (Home, Now Playing, Craft Quiz) using React + TypeScript + Tailwind CSS. This is a pixel-approximation of the provided mobile screenshots.

## What’s included
- Home screen (XP card, actions, story carousel)
- Now Playing (audio player UI)
- Craft a Custom Quiz (topic + difficulty)
- Simple navigation simulation
- Tailwind theming for green/mint colors similar to screenshots
